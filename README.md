# 🚀 Rakshak AI – Smart Soldier Healthcare System

## 📌 Project Overview

Rakshak AI is an AI-based healthcare monitoring system designed for soldiers.
It helps in tracking health conditions, managing data, and providing real-time insights through a dashboard.

---

## 🧠 Features

* 🩺 Soldier health monitoring system
* 📊 Interactive dashboard (Frontend)
* 🔗 REST API integration (Backend)
* ⚡ Real-time data handling
* 🛡️ Scalable architecture

## ⚙️ Installation & Setup

### 🔹 1. Clone the repository

```
git clone https://github.com/NitanGanutra/Rakshak-AI-.git
cd Rakshak-AI-
```

---

### 🔹 2. Backend Setup

```
cd rakshakai/backend
npm install
node server.js
```

---

### 🔹 3. Frontend Setup

(Open new terminal)

```
cd rakshakai-dashboard
npm install
npm start
```

---
